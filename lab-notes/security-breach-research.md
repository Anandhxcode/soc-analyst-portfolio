# Security Breach Research

## 1. Equifax Breach (2017)

- **What was taken:**
  Personal information and Social Security numbers

- **Exploit used:**
  Unpatched Apache Struts vulnerability

- **Prevention:**
  Regular patch management and vulnerability scanning

---

## 2. Facebook Data Leak (2021)

- **What was taken:**
  User phone numbers and profile data

- **Exploit used:**
  Data scraping vulnerability

- **Prevention:**
  Better API security and rate limiting

---

## 3. Twitter Hack (2020)

- **What was taken:**
  Verified Twitter account access

- **Exploit used:**
  Social engineering attack

- **Prevention:**
  Multi-factor authentication and employee training

---

## 4. MGM Resorts Cyberattack (2023)

- **What was taken:**
  Customer data and system access

- **Exploit used:**
  Help desk social engineering and ransomware

- **Prevention:**
  Identity verification and network segmentation

---

## 5. Colonial Pipeline Attack (2021)

- **What was taken:**
  Operational systems disrupted

- **Exploit used:**
  Compromised VPN credentials and ransomware

- **Prevention:**
  Enable MFA and strong password policies
